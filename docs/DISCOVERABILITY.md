# QakeAPI Discoverability

Help others find QakeAPI. Here's how to add it to popular lists.

## Awesome Python

[awesome-python](https://github.com/vinta/awesome-python) is a curated list of Python frameworks and libraries.

**How to add QakeAPI:**

1. Fork [awesome-python](https://github.com/vinta/awesome-python)
2. Add under **Web Frameworks** or **WebSocket**:
   ```markdown
   - [QakeAPI](https://github.com/craxti/qakeapi) - Zero-dependency hybrid sync/async web framework with OpenAPI, WebSocket, DI.
   ```
3. Open a Pull Request

## PyPI

QakeAPI is published on [PyPI](https://pypi.org/project/qakeapi/). Keywords and description are set in `setup.py` for search optimization.

## GitHub Topics

**Add these topics** to improve discoverability in GitHub search:

1. Go to your repo: https://github.com/craxti/qakeapi
2. Click the **⚙️ gear icon** next to "About" (or "Add topics")
3. In the Topics field, add (comma-separated or one by one):

```
python
web-framework
asgi
api
rest
openapi
websocket
async
zero-dependencies
```

4. Click **Save changes**

*Topics are keywords that help others find your project when searching GitHub.*
